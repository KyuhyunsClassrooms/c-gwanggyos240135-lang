오류 1: 단어 개수가 50개를 넘을 때 문제 발생
원인: 최대 개수 검사 없음
수정: if (count >= MAX) 조건 추가
결과: 범위 초과 입력 방지됨

오류 2: 단어가 없는데 목록 보기 실행
원인: count == 0 체크 없음
수정: 목록 보기 전에 0인지 확인하도록 변경
결과: "저장된 단어 없음" 정상 출력

오류 3: 잘못된 메뉴 번호 입력
원인: 메뉴 번호에 대한 예외 처리 부족
수정: else로 “잘못된 입력” 출력
결과: 프로그램 흐름 정상 유지

 디버깅 결과
모든 기능(추가, 목록, 종료)이 정상 동작하며,
입력 오류에 대한 기본적인 예외 처리가 완료됨.

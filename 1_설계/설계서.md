1. 프로그램 목적
영어 단어와 뜻을 입력받아 단어장에 저장하고, 저장된 단어 목록을 확인할 수 있는 간단한 프로그램이다.

2. 주요 기능
단어 추가
영어 단어와 한국어 뜻 입력
배열에 저장
단어 목록 보기
저장된 단어를 번호와 함께 출력
프로그램 종료

3. 데이터 구조
typedef struct {
    char eng[30];
    char kor[30];
} Word;
Word words[50];   // 최대 50개 저장
int count = 0;    // 저장된 단어 수

4. 처리 흐름
메뉴 출력
1: 단어 추가
2: 단어 목록 보기
0: 종료
입력에 따라 기능 수행
종료 선택 시 프로그램 종료
5. 사용 문법
구조체
배열
반복문(while, for)
조건문(if)
표준 입력/출력 (scanf, printf)
